# 🌐 CN API Tester

Professional REST API testing tool built with modern web technologies.

Test, inspect and analyze API endpoints directly in your browser with support for multiple HTTP methods, formatted JSON responses and real-time performance metrics.

## ⚡ Getting Started

Clone the repository:

```bash
git clone https://github.com/ChrisNo86/cn-api-tester.git
```

Open:

```text
index.html
```

in your browser.

No installation or dependencies required.
